ABOUT US

We aim to create the best ski event that no one can surpass


皆さんはどんなスキー行事にしたいですか？こういうのってありきたりなものになってしまいません？ 「みんなが楽しめる行事にする」という目標があります。これ、どの目標よりも難しいんです。皆さんも感じていると思います。「みんな」が納得して「みんな」が楽しめるもの。そんなもの創れるはずがないんです。でも！もしそれが実現して「みんな」が楽しむ事が出来たら？考えただけでにやけてきます。 今、私たちが不可能な事に現実を引っ張り寄せている途中です。私たちがこの3年間で培ってきたすべての力を出せば必ず出来ます。 全速前進DA！We aim to create the best ski event that no one can surpass
